Future Ideas...

Joe's future aspirations lie in Web & App development. Specifically, creating an all in one application for locating music & vocal tutors and viewing/curator content. Think Google Maps meets Patreon, with some unique USP's.

Proposed Methodology: Using Python Automation to scan nearby Tutors, using a Database language (probably NoSQL) to store the data; finally, presenting the finding in a web browser application & mobile app. This is hoped to evolve into an application where Tutors can upload content, similar to patreon, but then offer contact details & directions to their studios. The main USP would be the ability to record and view content in VR or Augmented reality. The main setback for this is that it requires a good amount of content creators and community engagement for insturment & vocal music learners. When technology evolves to allow this in real-time. The goal is developing the archetecture to make this a national success when technology allows full functionality. This may be a great idea to re-visit and refine in the future.
